\doxysubsection{Detailed Description}
A pointer determining the position of the associated \mbox{\hyperlink{classkvs_1_1utils_1_1Value}{Value}} in the values file. Also stores information whether the associated value is present or deleted. 

The position is internally stored in a ptr\+\_\+t variable as the index of the \mbox{\hyperlink{classkvs_1_1utils_1_1Value}{Value}}. However, there are some other convenience methods to acces the data.
