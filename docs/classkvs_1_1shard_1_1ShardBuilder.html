Rebuild the Shard according to delayed removals stored in CacheMap.

The old shard is modified, but it is intended to be removed with a new Shard. The old Shard is disallowed to use after calling this method.

Parameters
shardThe Shard to rebuild.
cacheMapThe CacheMap to read information about delayed removals from.
Returns
pair.first - The newly created Shard to replace the old one.
pair.second - Entries in CacheMap that have to be overwritten, including both entries with new ptrs and old removed entries with EMPTY_PTR.
