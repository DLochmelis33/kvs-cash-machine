Enumeration Type Documentation

◆ PtrType

enum kvs::utils::PtrType
strong

A convenience class to check Ptr's type.

PRESENT - this Ptr points to an actual value DELETED - this Ptr points to a value that existed before, but now is deleted from KVS SYNC_DELETED (deprecated, see NONEXSITENT) - this Ptr points to nowhere, was lazily deleted in CacheMap and then synced with the Shard NONEXISTENT - this Ptr points to nowhere, but it was PRESENT before and the current corresponding Key is valid EMPTY_PTR - this Ptr is equivalent to NULL

NONEXISTENT is forbidden in StorageHashTable
